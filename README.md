Este proyecto es el juego Pac-man llevado a Javascript.

# Comandos

```bash
# Inicia el servidor de desarrollo en localhost:3000
npm start

# Watcher en modo desarrollo
npm run watch

# Compila assets en modo productivo
npm run build
```
